dmin Dashboard with Auth & Validation

 A high-quality, professional Admin Panel featuring a secure authentication flow and dynamic form validation with custom error messages.

  Key Features
- **Authentication**: Fully functional Login and Signup pages.
- **Real-time Validation**: Text-based error messages for email format, password length, and empty fields (No annoying alerts!).
- **User Interface**: A clean and modern Admin Dashboard layout.

  Tech Stack
- **HTML5**: Semantic forms and dashboard structure.
- **CSS3**: Custom UI, Flexbox/Grid, and error styling.
- **JavaScript (ES6)**: DOM manipulation for validation logic and navigation.
